What You’ll Learn

From foundations to hands-on installations: design, install, configure and run solar-inverter systems.

Module 1: Intro to Solar Energy

Types of systems, solar radiation basics and how panels generate electricity.

Module 2: Solar System Design

Load calculations, panel/inverter/battery sizing and layout planning.

Module 3: Inverter Installation

Rack, wire, configure and commission inverters for resilient systems.

Module 4: Battery Management

BMS basics, battery configurations, safety and longevity strategies.

Module 5: Safety Standards

Codes, earthing, PPE, and professional installation best practices.

Module 6: Business Setup

Quoting, client acquisition, project management and post-sales support.
